Python Intro

what is Python

Python is a versatile and widely-used programming language known for its simplicity and readability, making it an excellent choice for beginners and experienced developers alike. Created by Guido van Rossum and first released in 1991, Python emphasizes code readability and allows developers to write clear and logical code for both small and large-scale projects.

Key Features of Python

  • Readability
  • Interpreted Language
  • Dynamically Typed
  • High-Level Language
  • Extensive Standard Library
  • Community and Ecosystem

Installing Python on Windows

Download the Installer:

  • Go to the official Python website: python.org.
  • Navigate to the "Downloads" section and download the latest Python release for Windows.

Run the Installer:

  • Locate the downloaded file (python--amd64.exe or similar) and run it.
  • Check the box that says "Add Python to PATH".
  • Click on "Install Now".

Verify Installation:

  • Open Command Prompt by typing cmd in the search bar and pressing Enter.
  • Type python --version and press Enter. You should see the Python version you installed.

Installing Python on macOS

Download the Installer:

  • Go to the official Python website: python.org.
  • Navigate to the "Downloads" section and download the latest Python release for macOS.

Run the Installer:

  • Locate the downloaded .pkg file and run it.
  • Follow the instructions in the installer.

Verify Installation:

  • Open Terminal by searching for it in Spotlight or from the Applications folder.
  • Type python3 --version and press Enter. You should see the Python version you installed.


Python Syntax

Python syntax refers to the set of rules that defines how a Python program is written and interpreted. Python's syntax is designed to be readable and straightforward.

    Key Points:
  • Python uses indentation to define blocks of code.
  • Statements end with a newline, not a semicolon.
  • Variables are created when they are first assigned values.

In this example, the if-else statement uses indentation to define the blocks of code. Python does not use braces { } to delimit blocks of code like some other languages.

Python Comments

Comments in Python are used to explain code and make it more readable. They are ignored by the Python interpreter. There are two types of comments: single-line and multi-line comments.

Single-Line Comments:

Single-line comments start with a # symbol.

Multi-Line Comments:

Multi-line comments can be written using triple quotes ''' or """.

                        
# This is a single-line comment
"""
This is a multi-line comment
spanning multiple lines.
"""
def add(a, b):
    # This function adds two numbers
    return a + b
result = add(5, 3)
print("The result is:", result)
Copied!

In this example, the comments explain what the code is doing. The single-line comment describes the purpose of the add function, and the multi-line comment provides a general description.


Python Variables

Variables in Python are used to store information that can be referenced and manipulated in a program. Unlike other programming languages, Python variables do not need explicit declaration to reserve memory space. The declaration happens automatically when a value is assigned to a variable.

                        
x = 5
y = "Hello, Python!"
print(x)
print(y)
Copied!

variable names

Variable names in Python must adhere to certain rules:

  • Must start with a letter or the underscore character.
  • Cannot start with a number.
  • Can only contain alpha-numeric characters and underscores (A-z, 0-9, and __).
  • Are case-sensitive (e.g., myVar and myvar are different variables).
                        
# Valid variable names
my_var = 10
myVar = 20
_my_var = 30
myVar2 = 40
print(my_var, myVar, _my_var, myVar2)

# Invalid variable names (uncommenting these will cause errors)
# 2myvar = 50
# my-var = 60
# my var = 70

The example shows valid variable names and some examples of invalid ones.

Assign Multiple Values

Python allows you to assign values to multiple variables in one line. This is particularly useful for initializing multiple variables quickly.

                        
# Assigning multiple values to multiple variables
a, b, c = 5, 10, 15
print(a)
print(b)
print(c)
                            
# Assigning the same value to multiple variables
x = y = z = 20
                            
print(x)
print(y)
print(z)
Copied!

In the first example, the values 5, 10, and 15 are assigned to a, b, and c respectively. In the second example, the value 20 is assigned to x, y, and z.

Output Variables

You can use the print function to output variables. You can combine text and variables using commas or the + operator.

                            
# Outputting variables with the print function
name = "Alice"
age = 25
# Using commas
print("Name:", name)
print("Age:", age)
# Using the + operator (note the need for conversion to strings)
print("Name: " + name)
print("Age: " + str(age))
# Using f-strings (available in Python 3.6+)
print(f"Name: {name}")
print(f"Age: {age}")
Copied!

This example shows different ways to print variables along with text.

Global Variables

A global variable is one that is defined outside of a function and can be accessed inside any function within the same module. If you need to modify a global variable inside a function, you must declare it using the global keyword.

                            
# Global variable example
x = "global"
def my_function():
    global x
    x = "local"
    print("Inside function:", x)
                                
my_function()
print("Outside function:", x)
Copied!

Here, x is defined as a global variable. Inside the function, the global keyword is used to indicate that we are referring to the global x variable. Without the global keyword, the assignment would create a local variable named x.

By understanding these topics, you'll be well-equipped to work with variables in Python, including how to name, assign, and manipulate them effectively in your programs.

In Python, numeric data types are used to store numbers. There are three distinct numeric types: integers, floating-point numbers, and complex numbers.

Integer (int)

Integers are whole numbers, positive or negative, without decimals. They have unlimited precision in Python 3.

                            
a = 10
b = -5

Floating-point numbers are real numbers with a fractional part. They are specified with a decimal point.

                            
x = 10.5
y = -7.8

print(type(y))  # Output: class 'float'
print(type(z))  # Output: class 'float'
Copied!

Complex Number (complex)

Complex numbers are numbers with a real part and an imaginary part. They are written with a "j" as the imaginary part.

                            
p = 3 + 5j
q = -2j
r = 7j
print(type(p))  # Output: class 'complex'
print(type(q))  # Output: class 'complex'

Python Casting

Casting in Python is converting one data type into another. This is useful when you need to convert values between types, such as turning a string into an integer or a float into a string.

Integer Casting (int())

Casting to an integer discards the fractional part and rounds down to the nearest whole number.

                            
x = int(10.8)  # x will be 10
y = int("15")  # y will be 15
print(x)  # Output: 10
print(y)  # Output: 15
Copied!

Floating-Point Casting (float())

Casting to a float converts integers or strings into floating-point numbers.

                            
a = float(10)  # a will be 10.0
b = float("20.5")  # b will be 20.5
print(a)  # Output: 10.0
print(b)  # Output: 20.5
Copied!

String Casting (str())

Casting to a string converts numbers or other types into strings.

                            
i = str(123)  # i will be "123"
j = str(45.67)  # j will be "45.67"
print(i)  # Output: 123
print(j)  # Output: 45.67
Copied!

Practical Examples of Casting

Combining different casts in practical situations:

                            
# Convert user input into an integer
user_input = input("Enter a number: ")
num = int(user_input)
print("The number you entered is:", num)
print("Type of the number:", type(num))
                                
# Convert a float result into an integer
result = float(9.99)
print("Float result:", result)  # Output: 9.99
result = int(result)
print("Converted to integer:", result)  # Output: 9
Copied!

In this example, input() returns a string, so we cast it to an integer using int(). Similarly, a floating-point result is cast to an integer, which truncates the decimal part.

By understanding Python Numbers and Casting, you can handle various types of numerical operations and convert between data types as needed in your programs. This is especially useful when dealing with user input, mathematical computations, and data processing tasks.

Python Strings

Strings in Python are sequences of characters enclosed in single quotes ('), double quotes ("), or triple quotes (''' or """). Strings are immutable, meaning once they are created, their contents cannot be changed.

                            
single_quote_str = 'Hello'
double_quote_str = "World"
triple_quote_str = """This is a
multiline string"""
                                
print(single_quote_str)  # Output: Hello
print(double_quote_str)  # Output: World
print(triple_quote_str)  # Output: This is a\nmultiline string
Copied!

Slicing Strings

String slicing allows you to obtain a substring by specifying a range of indices. The syntax is string[start:end:step].

                            
text = "Hello, World!"

# Slicing to get 'Hello'
print(text[0:5])  # Output: Hello
                                
# Slicing to get 'World'
print(text[7:12])  # Output: World
                                
# Slicing with a step
print(text[::2])  # Output: Hlo ol!
Copied!

Modify Strings

Since strings are immutable, modifying a string means creating a new string. This can be done using concatenation, slicing, and other methods.

                            
text = "Hello, World!"

# Converting to uppercase
new_text = text.upper()
print(new_text)  # Output: HELLO, WORLD!
                                
# Replacing characters
new_text = text.replace("World", "Python")
print(new_text)  # Output: Hello, Python!
                                
# Adding a substring
new_text = text[:7] + "Beautiful " + text[7:]
print(new_text)  # Output: Hello, Beautiful World!
Copied!
